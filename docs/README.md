# docs — Project Documentation

Supporting documentation for the IIoT OI Platform.

## Contents

### `internal/`
`project-context.md` — full project context used to brief AI assistants
when continuing work across sessions. Contains background, architecture,
credentials, and current status. Not intended for public consumption.

### `screenshots/`
Captures of the live system: Grafana dashboard panels, Airflow DAG runs,
EMQX broker dashboard, Supervisor process list. Useful for the project
report and for sharing on LinkedIn or in job applications.

## Planned additions

- **Project report (PDF)** — a detailed technical narrative covering the
  problem statement, architecture decisions, implementation, debugging
  journey, and lessons learned. The deep-dive document for hiring managers
  who want more than the README.
- **Architecture diagram** — a clean, exportable diagram of the full
  data pipeline suitable for embedding in presentations or the report.
